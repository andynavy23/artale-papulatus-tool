# artale-papulatus-tool
tool for papulatus difficult mode

## 使用說明
- 點擊左側選單中的 連結遊戲畫面 按鈕
- 調整 即時監控畫面 中的 偵測框 位置與大小
  - 血量偵測框：框住拉圖斯血量 數字與趴數 即可
  - 對話偵測框：框住對話框中最後一行 新對話會出現的地方即可
  - 小地圖偵測框：框住小地圖即可
- 調整 小地圖預覽 中的 標記線 位置 (參考左側安全點位置調整)
- 根據 小地圖預覽預覽 畫面中檢查是否顯示綠色準心(玩家位置)
    - 有顯示綠色準心代表 RGB 設定正確
    - 沒有顯示綠色準心代表 RGB 設定需要調整, 請調整 RGB 設定 中的 R、G、B 數值與誤差容許值，直到小地圖預覽中的準心顯示出來為止 
- 點擊儲存設定按鈕
